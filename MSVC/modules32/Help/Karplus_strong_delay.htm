Karplus-Strong Delay

This is a variable-length delay-line with a simple low-pass filter in the feedback path, which can be excited with a short noise burst at the Signal In input, to implement the Karplus-Strong plucked-string algorithm.

The Pitch input interprets a standard SynthEdit pitch-control voltage value (0.0 to 10.0V, scaled to range 0.0 - 1.0, 5V = 440 Hz, 1 Volt per octave) to set the length of the internal delay-line to exactly one cycle of the selected pitch.

Pins

  • Signal In excitation signal
  • Pitch pitch CV, as for any SE oscillator
  • Feedback normally leave this at the default value 1.0 (100%)
  • Signal Out audio output
